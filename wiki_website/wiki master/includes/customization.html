# customization

  • override the attribute data-layout in div id="app" data-layout=".." to choose a specific layout type
    this attribute accepts the values fwidth, boxed
  • override the attribute data-display in div id="app" data-display=".." to choose a display type
    this attribute accepts the values slides, one-page, minimal
  • override the attribute data-theme in div id="app" data-theme=".." to choose a theme color
    this attribute accepts the values amber, cyan, emerald, lime, orange, purple, red, rose.
  • override the attribute dir in body dir="rtl" to set direction for the template
    this attribute accepts the values rtl, ltr.
  • add the class dark to body so you can set dark mode as default mode
  • add the class main-bg-color to any element so you give it the choosen theme color's background color
  • add the class main-text-color to any element so you give it the choosen theme color's text color
  • add the class main-border-color to any element so you give it the choosen theme color's border color
