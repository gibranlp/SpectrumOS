# installation and use

how to install
  • unzip the product file
  • browse the product files using a code editor such as VS code
  • create a new .HTML file that follows the structure described bellow
  • choose a proper customization for your layouts following the tips described bellow in customization section
  • start using the code snippets from includes directory to document your product
