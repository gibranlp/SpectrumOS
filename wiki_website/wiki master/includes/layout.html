Table of contents
Important the table of contents items are auto-generated from the current active section
so you don't need to descibe any content inside ul inside the table of contents
any items inside any section that has the class bellow will be considered as a content item for that section
content-item 
<!-- table of contents starts here ///////////////////// -->
<div class="table-of-contents" id="table-of-contents">
    <h2 class="title text-lg font-bold" data-display-title="any text here will be diplayed as table of contents title"></h2>
    <ul class="w-full mt-5">
       <!-- leave blank //////////////////// -->
    </ul>
</div>
<!-- end of table of contents ///////////////////// -->

in this item for example, base will considered as content item and automatically previewed in the table of contents section

<div class="content-item section-description"> 
    Base
</div>
sidebar
  • any list item that contains a nested list should have an anchor that has the class has-nested 
    <li>
  <a href="#" class="has-nested">
      ..
  </a>
</li>
  • any nested lists should have the class nested 
    <li>
    <ul class="nested">
       <li>
          <a href="#">
              ..
          </a>
       </li>
    </ul>
</li>
sections
  • any section can be structured as bellow 
    <section id="sectionId">
    <h2 class="section-title"> 
        <a href="#sectionId">#</a>
        section title here
    </h2>
    
    <div class="section-description">
        section description here
    </div>
    
    <div class="section-content">
        section content here
    </div>
</section>
